Privacy Policy

Portable Puzzle Collection for Windows Store is an offline single-player game, designed to run without connecting to any network or service.

Portable Puzzle Collection does not require any permissions or capabilities. This means that the app cannot access the internet or hardware functions. The app collects no personal information.

The app can access saved games (.puzzle files) when opened by the user, either through Windows Explorer or by using the Load Game option in the app. Saved games can be transmitted by the user with the Save Game or Share Game options in the app. The app doesn't read any files that were not explicitly selected by the user.

Installation through the Microsoft Store is subject to Microsoft's privacy policy.
