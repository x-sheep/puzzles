Portable Puzzle Collection for Windows Store

Port of Simon Tatham's Portable Puzzle Collection to C++ Windows Store and Windows Phone apps. The interface is built in XAML, and the graphics are drawn with Direct2D.


Available for Windows 10 desktops and tablets. Older versions are available for Windows 8.1.

View source on GitHub

List of puzzles

Black Box, Bridges, Cube, Dominosa, Fifteen, Filling, Flip, Flood, Galaxies, Guess, Inertia, Keen, Light Up, Loopy, Magnets, Map, Mines (Minesweeper), Mosaic, Net, Netslide, Palisade, Pattern, Pearl, Pegs (Solitaire), Range, Rectangles, Same Game, Signpost, Singles, Sixteen, Slant, Solo (Sudoku), Tents, Towers, Train Tracks, Twiddle, Undead, Unequal, Unruly, Untangle

Features

  • Virtual keyboard for touch control
  • Colorblind setting for Flood, Guess, Map and Same Game puzzles
  • Save games to harddisk, OneDrive or other service
  • Send saved games using the Share charm
  • Cancel button during puzzle generation

Older versions of Windows

Windows 8.1 and Windows Phone will no longer receive official builds, but it's still possible to compile them yourself with Visual Studio 2015. See the repository for more information.
